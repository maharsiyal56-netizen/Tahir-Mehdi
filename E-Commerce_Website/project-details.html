How to Run the Project

1

Download Files

Download all the HTML, CSS, JS, and image files into a single folder named "techstore-ecommerce"

2

Open in Browser

Open the index.html file in any modern web browser (Chrome, Firefox, Edge, Safari)

3

Explore the Website

Navigate through all pages using the header menu. Test the shopping cart functionality by adding products to cart and proceeding to checkout

4

Local Development

For development, use a code editor like VS Code. The project runs completely locally with no server requirements
